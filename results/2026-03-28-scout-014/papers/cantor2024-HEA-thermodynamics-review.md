## Key Findings (from review literature)

**Configurational entropy definition:**
ΔS_conf = -R Σᵢ xᵢ ln(xᵢ)
This is IDENTICAL to Shannon entropy H = -Σ pᵢ log(pᵢ) in information theory and ecological diversity indices.

**HEA classification criterion:** ΔS_conf > 1.5R (equivalent to Shannon entropy H > 1.5 nats)

**Hume-Rothery rules for solid solutions:**
1. Atomic radius difference < 15%
2. Same crystal structure
3. Similar electronegativity
4. Same valence

**Beyond entropy:** Multiple comprehensive studies (2023-2025) show configurational entropy alone is insufficient for phase prediction — mixing enthalpy (pairwise interactions) critically determines whether single-phase solid solution forms.

**Counter-example problem:** Alloys satisfying entropy thresholds but forming intermetallics; alloys violating thresholds yet producing single phases.
