# The Mpemba Index and Anomalous Relaxation

**Authors:** Israel Klich, Oren Raz, Ori Hirschberg, Marija Vucelja
**Year:** 2019
**Journal:** Physical Review X
**DOI:** https://doi.org/10.1103/PhysRevX.9.021060
**arXiv:** 1711.05829

## Abstract Summary
The Mpemba effect is a counter-intuitive relaxation phenomenon where a system prepared at higher temperature cools faster than one at a lower starting temperature when both are quenched to the same cold bath. The authors formalize this using Markovian dynamics.

## Key Findings

**Mpemba Index:** The number of special initial temperatures exhibiting the "strong" Mpemba effect, where relaxation time jumps to a smaller value enabling exponentially faster equilibration. The parity of this index is a topological property of the system.

**Spectral Mechanism:** The strong effect arises when a "carefully chosen set of initial temperatures" causes the overlap with the slowest-decaying eigenmodes of the Liouvillian to vanish. The system then relaxes via faster eigenmodes only — exponentially speedup.

**Non-monotonic temperature decay:** The effect is connected to thermal overshoot — non-monotonic temperature during relaxation — analyzed in mean-field anti-ferromagnetic Ising models.

**Generality:** The effect survives in thermodynamic limits and occurs with non-zero probability across diverse thermal quench processes. The Mpemba phase diagram is "surprisingly rich."
