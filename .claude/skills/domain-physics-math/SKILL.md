# Physics & Mathematics Discovery Guide

## High-Potential Connection Zones (updated 2026)
- Shared mathematical structures across different physical systems
- Symmetry breaking analogues across scales
- Information-theoretic formulations of physical problems
- Topological properties ↔ measurable physical quantities
- Computational complexity ↔ physical phase transitions
- Renormalization group ↔ hierarchical abstraction in other domains
- Quantum error correction ↔ biological robustness
- Continuous mathematics tools applied to discrete algorithmic problems
  (cf. Aletheia resolving Erdős problems, Feb 2026)

## Critical Distinction: Analogy Depth
- **Surface analogy** (LOW value): Same word, different structures
- **Structural correspondence** (HIGH value): Same formal properties
- **Formal isomorphism** (HIGHEST value): Literally same mathematical object

Only structural correspondence and formal isomorphism are scientifically productive.

## Key Search Patterns
- `site:arxiv.org [query]`
- MathSciNet for mathematical results
- INSPIRE-HEP for particle physics

## Common Pitfalls
- Formal similarity ≠ physical relevance
- Mathematical elegance ≠ physical truth
- Dimensional analysis: always check units
- Scale matters: quantum effects rarely matter at macro scale
